# Bolt.new — Bolt Cloud Analysis
> Source: support.bolt.new/cloud/bolt-cloud (scraped 2026-03-30)

## Overview
Bolt Cloud = all-in-one backend infrastructure. Powered by Netlify (hosting) and Supabase (database).

## Components

### Hosting
- Auto-hosted on secure, high-performance servers
- Free `.bolt.host` subdomain for all users
- Custom domains for paid users
- Share (restricted) or Publish (public) options
- Nothing to configure

### Databases
- **Unlimited databases** per project
- Auto-created when app needs one
- Table view editor for data
- Activity logs, edge functions, secrets management
- User management and authentication built-in
- Can claim database and connect to Supabase

### Domains
- Buy new domains directly in Bolt
- Connect existing domains
- No DNS editing required for Bolt-purchased domains
- Goes live in few clicks

### Authentication & User Accounts
- Signups, logins, password resets built-in
- Roles and permissions
- Handled securely behind the scenes

### File Storage
- Store and share files (images, documents, uploads)
- View all files in one place
- No separate storage systems needed

### Server Functions (Edge Functions)
- Backend code execution
- Forms, data processing, custom features
- Runs close to users for quick response

### Analytics
- Pageviews, visitors, traffic sources
- Built-in, no extra scripts or accounts

### Payments
- Stripe integration for one-time or subscriptions
- Security and compliance handled by Bolt Cloud + Stripe

## Key Observations for ArkhosAI
- "No separate accounts" is the key selling point — all-in-one
- Netlify + Supabase under the hood — proven infrastructure
- Unlimited databases is aggressive positioning
- No mention of data residency or EU hosting options
- No GDPR-specific controls or data processing mentions
- Analytics built-in is smart — removes need for external tool
