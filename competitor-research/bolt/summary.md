# Bolt.new — Key Findings Summary

## Strengths
1. **All-in-one platform:** Bolt Cloud (hosting + DB + auth + domains + analytics + edge functions)
2. **100% Claude-powered:** strongest coding model family, multiple tiers available
3. **Plan Mode from homepage:** users can plan before building, reducing waste
4. **Figma import:** strong design-to-code pipeline
5. **Expo for mobile:** actual mobile app support via React Native
6. **Unlimited databases:** no DB limits is aggressive
7. **MCP support:** Notion, Linear, GitHub, custom servers
8. **WebContainers** (StackBlitz): browser-based dev environment, no server needed
9. **Import from Lovable:** actively competing for users
10. **Design system support:** on-brand building

## Weaknesses
1. **JavaScript-only backend** — no Python, PHP, Go, Rust
2. **Token pricing is opaque** — no per-token or per-model cost visibility
3. **No data residency controls** — no EU/GDPR-specific hosting
4. **No security certifications mentioned** (no SOC 2, ISO 27001, GDPR claims)
5. **Mobile browser not supported** — desktop only
6. **v1 Agent retirement** — forces migration, could lose users
7. **No model routing** — user manually selects Claude model
8. **No cost/budget controls** — can't set spending limits
9. **Token rollover complexity** — 2-month expiry creates confusion
10. **Fewer integrations than Lovable** — no Slack, Shopify, Telegram, etc.

## What ArkhosAI Can Learn
- Bolt Cloud "all-in-one" is the right UX — minimize external accounts
- Plan Mode from the first prompt is smart
- Figma import is table stakes
- WebContainers for browser-based dev is impressive tech
- Design system support is important for teams
- "Import from competitor" is a growth hack

## ArkhosAI Differentiators vs Bolt
- **Multi-language backend** — Python, JS, and more via Mistral code models
- **Mistral-native** — EU AI, no US dependency
- **Transparent pricing** — cost per call, per model, budget controls
- **Intelligent model routing** — automatic model selection, not manual
- **GDPR-first** — EU entity, EU hosting, proper data sovereignty
- **Security certifications** (planned) — SOC 2, ISO 27001
- **Open-core** — self-hostable option
