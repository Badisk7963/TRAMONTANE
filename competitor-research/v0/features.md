# v0 by Vercel — Features Analysis
> Sources: v0.app homepage, v0.app/pricing (scraped 2026-03-30)

## Core Features
1. Chat-based generation
2. Design Mode — visual editing with live preview
3. Template library (community-driven, some paid)
4. Design systems — colors, typography, styles across projects
5. GitHub sync — push code to repos
6. One-click Vercel deploy
7. Agentic by default — auto-planning, task creation, DB connection
8. iOS app — build from phone
9. File attachments in chat
10. Integration gallery

## AI Models (Custom v0 Models)
- v0 Mini — fast, near-frontier
- v0 Pro — balanced (default)
- v0 Max — maximum intelligence
- v0 Max Fast — max + 2.5x speed

## Tech Stack
- Frontend: React, Next.js, shadcn/ui, Tailwind CSS
- Deployment: Vercel (native)
- Version control: GitHub
- Design: shadcn/ui design system

## Vercel Platform Compliance
- SOC 2 Type 2, PCI DSS, ISO 27001
- EU-U.S. DPF, GDPR, HIPAA BAA (Enterprise add-on)

## Limitations
- Vercel-locked deployment
- Next.js/React focused
- Per-user pricing (expensive for teams)
- No mobile app building (only responsive web)
- No self-hosting option
- Training opt-out requires $100/user plan
