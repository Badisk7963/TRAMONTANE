# v0 by Vercel — Key Findings Summary

## Strengths
1. Most transparent pricing — per-token costs published
2. Design Mode — visual editing, unique
3. Vercel ecosystem — instant deploy, CDN, edge, analytics
4. Agentic by default — auto planning + task creation
5. iOS app — unique mobile creation tool
6. shadcn/ui — popular design system integration
7. Strong compliance — SOC 2, ISO 27001, PCI DSS, GDPR, HIPAA
8. Custom model tiers for user choice
9. Large template marketplace
10. Enterprise features (SSO, RBAC, SLAs)

## Weaknesses
1. Per-user pricing — most expensive for teams
2. Vercel lock-in — hard to deploy elsewhere
3. Next.js/React focused
4. Training opt-out only on Business+ ($100/user)
5. 7 msg/day free limit — most restrictive
6. Model opacity — hides actual AI model
7. Hosting costs on top of v0
8. No MCP support mentioned
9. No EU data residency for v0
10. No mobile app building

## ArkhosAI Differentiators
- EU-first GDPR native
- Mistral-native, transparent model identity
- Per-workspace pricing (dramatically cheaper)
- Automatic model routing
- Framework agnostic, deploy anywhere
- Self-hostable open-core
