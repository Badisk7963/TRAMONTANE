# Bolt.new — Supported Technologies
> Source: support.bolt.new/building/supported-technologies (scraped 2026-03-30)

## Browser Support
- Best in Chrome and Chromium-based browsers (Edge, Brave, Vivaldi, Opera, Epic)
- Mobile browsers NOT fully supported
- Ad blockers/privacy tools/VPNs can cause issues

## Platforms
- Websites
- Web apps
- Mobile apps via **Expo** integration

## Languages & Frameworks
- **JavaScript-based only**
- Node.js for backend
- Any JavaScript framework for frontend (React, Vue, Next.js, etc.)
- **No PHP, Python, or non-JS backends**

## Databases
- Bolt Cloud built-in database
- Supabase connection option

## Key Limitations for ArkhosAI
- **JavaScript-only backend** is a significant limitation
- No Python, no Rust, no Go — limits enterprise appeal
- Mobile only via Expo — no native mobile
- Desktop browser only — no mobile IDE
- This is a huge opportunity for ArkhosAI with Mistral's code models
